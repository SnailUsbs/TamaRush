# Tamarush Source Code

This is the source code behind Tamarush. 


## Required Software:
- DOTNET 4.7.2 (AKA: .Net 4.7.2)

## Required Libs To Build From Source:
- Assembly-CSharp.dll
- 0Harmony.dll
- BepInEx.dll
- Unity.TextMeshPro.dll
- UnityEngine.ImageConversionModule.dll
- UnityEngine.UI.dll
- CommonAPI.dll (BRC Back bone for custom apps)
-
- *All required libs are setup to be read from a folder named "lib" in the root of the project*

----------------------

## Feature Credit Breakdown:

- **Tamalib/Tamatool:** Both are made by the same person, and are what almost everything in the tamarush emulator is based off of.

- **BRC-CODE-DMG:** Delta transform for displaying the emulator, and BRC-CODE-DMG's entire emulator to BRC audio system.

- **Tamalib:** https://github.com/jcrona/tamalib

- **Tamatool:** https://github.com/jcrona/tamatool/

- **BRC-CODE-DMG:** https://github.com/AbsentmindedGCN/BRC-CODE-DMG/
